### **What Each Pipeline Does**

#### **ClinVar Pipeline**
- Downloads variant data from NCBI ClinVar
- Processes clinical significance annotations
- Loads variants with pathogenicity scores
- Creates indexes for fast variant lookup

#### **TCGA Pipeline**
- Connects to GDC API for cancer data
- Downloads clinical, expression, and mutation data
- Processes multi-omics data integration
- Creates sample-variant relationships

#### **PubMed Pipeline**
- Searches PubMed for genomics literature
- Extracts entities (genes, diseases, drugs)
- Builds citation networks
- Processes abstracts for text mining
